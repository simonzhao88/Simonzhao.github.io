什么是HTML?

HTML 是用来描述网页的一种语言。

  1. Tag: Content 装内容装数据
  2. CSS: Display 渲染,显示
  3. JavaScript: Behavior 交互,行为
  • HTML 指的是超文本标记语言: Hyper Text Markup Language
  • HTML 不是一种编程语言，而是一种标记语言
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页
  • HTML 文档包含了HTML 标签及文本 内容
  • HTML文档也叫做web 页面
